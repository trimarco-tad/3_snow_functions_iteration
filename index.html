Simple web scraping

R can read html using either rvest, xml, or xml2 packages. Here we are going to navigate to the Center for Snow and Avalance Studies Website and read a table in. This table contains links to data we want to programatically download for three sites. We don’t know much about these sites, but they contain incredibly rich snow, temperature, and precip data.

Reading an html

Extract CSV links from webpage

site_url <- 'https://snowstudies.org/archived-data/'

#Read the web url
webpage <- read_html(site_url)

#See if we can extract tables and get the data that way
tables <- webpage %>%
  html_nodes('table') %>%
  magrittr::extract2(3) %>%
  html_table(fill = TRUE)
#That didn't work, so let's try a different approach

#Extract only weblinks and then the URLs!
links <- webpage %>%
  html_nodes('a') %>%
  .[grepl('24hr',.)] %>%
  html_attr('href')

  1. Write a function that makes line plots of monthly average temperature at each site for a given year. Use a for loop to make these plots for 2005 to 2010. Are monthly average temperatures at the Senator Beck Study Plot ever warmer than the Snow Angel Study Plot? Hint: https://ggplot2.tidyverse.org/reference/print.ggplot.html
monthly_temp <- function(data = meteor_data_full){

  meteor_monthly <- meteor_data_full %>%
    group_by(year,month,site) %>%
    summarize(mean_temp = mean(`air temp [K]`,na.rm=T))
  
  meteor_monthly$year <- as.numeric(meteor_monthly$year)

for(i in c(2005:2010)){
  
   gg_data <- meteor_monthly %>%
               filter(year == i)
   
    print(ggplot(gg_data,aes(x=month,y=mean_temp,color=site)) + 
          geom_point() + geom_line() +
          ggtitle(i) + ylab("Mean Air Temperature (K)") + xlab("Month") +
          ylim(min = 255, max = 290) +
          ggthemes::theme_few() + 
          ggthemes::scale_color_few())
  }
}

At no point from 2005-2010 is the monthly average temperature at the Senator Beck site higher than that at the Snow Angel site. Each year follows generally the same pattern of average monthly temperatures, with some variation year to year, particularly it seems in the spring and fall months. Notable, the difference between average monthly temperatures seems roughly consistent across months and across years. Neither site seems to be warming or cooling in relation to each other. While this data is limited, it does seem that the peak is widening slightly and moving further to the right, potentially indicating longer periods of summer temperatures, and that these periods are occurring slightly later in the year.
